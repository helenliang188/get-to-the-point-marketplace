# Get To The Point Marketplace

Codex plugin marketplace for `get-to-the-point`, a writing skill that rewrites communications so they are direct, conversational, evidence-led, and values-grounded.

## Install

Add the public marketplace:

```bash
codex plugin marketplace add helenliang188/get-to-the-point-marketplace
codex plugin add get-to-the-point@get-to-the-point-marketplace
```

Then start a new Codex thread and invoke:

```text
Use $get-to-the-point to rewrite this paragraph: ...
```
